# Overview

**Framer motion** is hands down the best library I've had the pleasure of using when it comes to animating in NextJS.

I have multiple examples on this portfolio, including the portfolio itself where you can see framer motion implemented into the backbone of the animations.

This particular example shows an animating toast button and list. The key importance of this example is that by deleting a toast in the middle, the list layout adjusts rather than jumping to position.
