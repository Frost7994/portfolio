# Overview

**Framer motion** has been my library of choice when it comes to animating in NextJS, from the simplistic to the complete abstract. One specific use case I came across and then later discovered on youtube was binding the scroll position of a page to specific animations which produced some very cool effects. I was writing sloppy wet code everywhere and needed a solution to create a reuseable hook that exposed the value I needed for the animations, and voila **useBoundedScroll** came along.

The main idea of this hooks is that it is provided some set of bounds and a reference object if you require, and utilising the bounds values it accepts, it returns a value which can be transformed to prouce any animation you need. The specific use case on this portfolio is the navbar. When a user scrolls 50px downwards it reduces the size and opacity of the navbar and on scrolling up it restores both these properties.

The hook is as follows:

```jsx
// state
import { useEffect, RefObject } from "react";

// utils
import { useTransform, useScroll, useMotionValue } from "framer-motion";
import { clamp } from "@/lib/utils/clamp";

export const useBoundedScroll = ({
  bounds,
  reference,
}: {
  bounds: number,
  reference?: RefObject<HTMLElement>,
}) => {
  // let bounds = 50;
  let { scrollY } = useScroll({
    container: reference ? reference : undefined,
    layoutEffect: false,
  });
  let scrollYBounded = useMotionValue(0);
  let scrollYBoundedProgress = useTransform(
    scrollYBounded,
    [0, bounds],
    [0, 1]
  );

  useEffect(() => {
    const updateMotionValue = (val: number) => {
      let prev = scrollY.getPrevious();
      let diff = val - (prev || 0);
      let newVal = scrollYBounded.get() + diff;

      scrollYBounded.set(
        clamp({
          number: newVal,
          min: 0,
          max: bounds,
        })
      );
    };

    let unsubscribe = scrollY.on("change", (current) =>
      updateMotionValue(current)
    );

    return () => unsubscribe();
  }, [scrollY, scrollYBounded, bounds]);

  return {
    scrollYBounded,
    scrollYBoundedProgress,
  };
};
```
